### Trying Sample
Download the sample.
    $ git clone git://github.com/parroty/angularjs_samples.git
    $ cd angularjs_samples

Install prerequisite component.
    $ gem install sinatra

Launch web server.
    $ ruby server.rb

Open the url with the browser
    $ open http://localhost:4567
